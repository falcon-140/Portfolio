## Built with

- Pure HTML, CSS, JavaScript — zero frameworks, zero dependencies
- Google Fonts (Syne, DM Mono, Instrument Serif)
- Hosted on GitHub Pages — free, fast, no server needed

---

## Features

- Custom animated cursor with lag effect
- Scroll-triggered fade-up animations
- Animated tech ticker strip
- Fully responsive (mobile + desktop)
- Project cards linked directly to GitHub repos
- Dark editorial design — high contrast, no fluff

---

## Run locally

No build step. No npm install. Just clone and open.

```bash
git clone https://github.com/falcon-140/portfolio
cd portfolio
# open index.html in your browser — that's it
```
